## Week 1 - Information Storage Before Computers
- Before computers, information was stored in physical forms like Cuneiform tablets and papyrus scrolls.
- The transition from oral traditions to written records allowed for more complex and lasting data storage.

## Week 2 - Representing text and numbers in computers
- Text is represented in computers using encoding systems like ASCII and Unicode, where each character has a numeric code.
- Numbers are processed in binary format (base-2), using only 0s and 1s to represent all values.

## Week 3 - Text Structure
- Structured text uses specific formats like Markdown or HTML to organize information logically.
- Proper text structure is essential for both human readability and for computers to parse data accurately.

## Week 4 - Typing and Searching
- Efficient searching involves using Boolean operators (AND, OR, NOT) to narrow down or expand results.
- Touch typing and keyboard shortcuts significantly increase productivity when creating and editing digital documents.

## Week 5 - Command Line Interfaces CLI
- A Command Line Interface (CLI) allows users to interact with a computer by typing text-based commands instead of using a mouse.
- Commands like `ls` to list files and `cd` to change directories are fundamental for navigating file systems via the terminal.
